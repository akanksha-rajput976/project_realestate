#  Real Estate Platform

A full-stack real estate web application that enables users to **buy, sell, and rent properties** with secure authentication and advanced search capabilities.  
The project focuses on real-world functionality, responsive design, and application reliability.

---

##  Project Overview
- **Project Type:** Full-Stack Web Application  
- **Duration:** Nov 2024 – Dec 2024  
- **Purpose:** To provide a seamless platform for property transactions with efficient search and secure user access.

---

##  Features
- User authentication using **JWT-based security**
- Buy, sell, and rent property functionality
- Responsive and user-friendly UI
- Advanced property search with dynamic filters:
  - Location
  - Price range
  - Property type
- Secure and reliable application flow

---
